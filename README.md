# httpsocks

A fast HTTP-via-SOCKS reverse proxy

```
Usage of httpsocks:
  -bind string
        Address to listen on (default "0.0.0.0:10443")
  -debug
        Enable debug output (default true)
  -proxy string
        SOCKS5 proxy address (default "localhost:9050")
  -upstream string
        Upstream host URL (default "https://ipinfo.io:443")
```
